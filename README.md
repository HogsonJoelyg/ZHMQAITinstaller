# MareQuest 汉化补丁简易安装指南 (QQ群: 632837008)

直接双击运行 `installer.exe` 。安装器一般能自动识别你的 Steam 游戏路径（如未识别，在 Steam 库中右键游戏 -> 管理 -> 浏览本地文件，复制地址栏路径后在安装器里点“浏览”粘贴进去即可）。确认路径正确后点击 **[安装]**，在弹出的 data.win 替换提示中选择 **[是]**，静待进度条走完提示成功，就成功了。

**注意事项与常见问题：** 本补丁不会影响存档进度，但因游戏引擎机制，旧存档里已生成的角色英文名无法溯及汉化，强烈建议开新档游玩以获得最佳体验。如遇到安装器更新失败，请先检查网络；若游戏官方后续更新导致补丁失效，请勿点击恢复原版，建议在 Steam 中卸载重装游戏本体后再重新打补丁。测试不全难免存在漏译或错译，任何 Bug 或版本失效问题均可进 QQ群 `632837008` 交流。
